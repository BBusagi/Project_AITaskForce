# Desktop App

Electron 桌面端放在这个目录下。

## 目标

- 启动后直接进入和 `Leader` 的对话窗口
- 右侧固定展示 `Team / Task / Chat` 三个子面板
- 用桌面形态验证对话优先的协作入口

```bash
cd Clients/Desktop
```
